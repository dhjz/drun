# drun - 项目管理器

一个简单易用的本地项目管理工具，帮助你快速启动和管理各类开发项目。

## 功能特性

- 📁 项目管理：添加、编辑、删除项目
- 🎯 项目类型支持：前端、后端、命令、其他
- ⚡ 快速启动：一键运行项目启动命令
- 🏷️ 标签分类：使用标签组织和筛选项目
- 🔍 搜索功能：快速查找项目
- 📦 导入导出：备份和恢复项目数据
- 🎨 视图切换：支持完整版和简洁版
- 📝 自定义命令：灵活配置启动命令

## 技术栈

- 后端：Go 1.24+
- 前端：Vue 3
- 数据库：本地 JSON 文件
- 前端嵌入：使用 Go embed 打包

## 快速开始

### 开发环境

要求：
- Go 1.24 或更高版本

### 运行项目

Windows：
```bash
go run main.go
```

Linux/Mac：
```bash
go run main.go
```

默认监听端口：8002

访问：http://localhost:8002

### 自定义端口

```bash
go run main.go -p 8080
```

## 使用说明

1. 点击「+ 添加项目」添加你的开发项目
2. 选择项目路径，系统会自动分析项目类型和启动命令
3. 配置自定义命令（可选）
4. 点击对应按钮即可快速启动项目
5. 使用标签、搜索、类型筛选来组织项目

## 数据存储

项目数据存储在本地 JSON 文件中，具体位置取决于运行环境。
